# test-vue-cli

## Project setup
```
npm install 
```

### What's installed ?
- Express server
- Nodemon
- Vuetify v2
- npm install vue-router@3
- axios (front)
- npm install cors (back)

## Start server
```
npm start
```

### Compiles and minifies for production
```
npm run build
```
